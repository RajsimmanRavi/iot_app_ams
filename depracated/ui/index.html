Kibana Dashboard

This is the dashboard for our platform. The graphs provides key insights on the statistics of the micro as well as macro services.

Basic Usage

This page is used to modify the configurations of the micro-services. You can modify the parameters such as Maximum Replication factors for services, CPU utilization limits etc.
Once you have performed the changes, you can reload the page to view the changes.
